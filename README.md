# Investment AI Navigator

AIを活用し、海外在住の日本人以外の投資家向けに市場分析・ニュース要約・投資判断の補助を提供するWebアプリケーションです。

---

## 🚀 Features

- AIによる市場分析
- 世界経済ニュースの自動収集
- 株式・ETF・為替・暗号資産の情報提供
- AIによる翌営業日の見通し
- ウォッチリスト
- パーソナライズされた投資情報
